## 常見決策主題

| 主題 | 注意事項 |
|---|---|
| MVC/Razor vs React | 團隊維護能力、互動複雜度、API 邊界。 |
| Console vs Python | 排程穩定性、套件環境、資料處理便利性。 |
| ArcGIS REST API vs Pro SDK | 是否依賴 ArcGIS Pro UI/GDB/FeatureClass/GP Tool。 |
| MSSQL schema 設計 | 查詢模式、索引、交易一致性、歷史資料。 |
| 排程與資料流 | 重跑策略、log、失敗補償、監控方式。 |
